## Purpose

Reusable auth service you can drop into any app. Demonstrates secure login/signup, refresh token rotation, password hashing, role-based access, input validation, tests, Docker, and CI.

### Tech stack
    .Node.js + TypeScript + Express
    .Prisma ORM + PostgreSQL
    .JWT (access + refresh tokens)
    .bcrypt
    .Zod for input validation
    .Jest + Supertest for tests
    .Docker, GitHub Actions
